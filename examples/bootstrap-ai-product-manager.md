# Example: Bootstrap AI Product Manager Learning Repo

User:

```md
领域：AI 产品经理
我的背景：做过运营，技术基础弱
每天可投入时间：60 分钟
目标：工作
最终作品：AI 产品 PRD + 原型说明
```

Expected behavior:

1. Create `learning-repo/`.
2. Write all P0 files.
3. Use the work goal adaptation.
4. Choose a small final project: AI feature PRD and prototype brief.
5. Make Day 1 about global understanding, product problem framing, and one-page project problem definition.
6. Mark current AI model/tool facts as needing latest verification.
7. Run `scripts/check_learning_repo.py learning-repo`.

Final reply shape:

```md
# 学习仓库已创建
## 仓库结构
## 已完整写入的核心文件
## 已创建的模板文件
```
