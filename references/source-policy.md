# Source Policy

Use this policy whenever the learning domain touches facts that may be current, high-risk, or externally verifiable.

## Safe To Generate Without Browsing

- general concept explanations
- learning tasks
- exercises
- self-test questions
- project decomposition
- reflection templates
- stable historical or conceptual summaries when uncertainty is low

## Must Mark As Needs Verification

Mark "不确定，需要核验最新来源" for:

- laws and regulations
- medical advice
- financial or investment conclusions
- policy changes
- software versions and API behavior
- product prices
- industry rankings
- company data
- market size
- news events
- academic paper details not provided by the user
- safety-critical procedures

## Investment Research Boundary

Allowed:

- explain frameworks
- identify risk factors
- build research memo structures
- design due-diligence questions

Not allowed:

- buy/sell recommendations
- fabricated financial data
- claims about current valuation without verification

## Medical, Legal, And Safety Boundary

Create learning material and questions, but do not give professional advice. Tell the user to verify with qualified sources or professionals when decisions have real-world consequences.

## Citation And Evidence Rule

If the user provides a source, name the source boundary. If browsing or current verification is not available, do not pretend to have checked the latest facts.
